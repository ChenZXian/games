# Archive Index

This directory stores superseded repository documents, retired workflow notes, and preserved historical assets.

Current archive areas:

- `bgm_legacy/` - older BGM workflow notes and migrated legacy BGM metadata
- `cursor_legacy/` - older Cursor rules versions
- `icon_legacy/` - older icon workflow notes, scripts, and source samples
- `misc/` - uncategorized retired files

Archive items are reference-only unless a newer active document explicitly points back to them.
